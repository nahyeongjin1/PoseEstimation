9. 모폴로지의 연산 종류에 대해서 적고, 각각에 대해서 설명하시오.

종류 : 침식, 팽창, 열림, 닫힘

1) 침식 연산
== 객체의 크기는 축소되고, 배경은 확장된다.
== 객체의 크기가 축소되기 때문에 영상 내에 존재하는 잡음 같은 작은 크기의 객체들은 사라질 수도 있다.

2) 팽창 연산
== 객체의 가장 외곽 화소를 확장시키기 때문에 객체의 크기는 확대되고 배경은 축소된다.
== 객체의 팽창으로 인해서 객체 내부에 있는 빈 공간도 메워지게 된다.

3) 열림 연산
== 침식 연산을 먼저 수행하고, 바로 팽창 연산을 수행한다.
== 침식 연산으로 인해서 객체는 축소되고, 배경 부분의 미세한 잡음들은 제거된다.
== 다음으로 축소되었던 객체들이 팽창 연산으로 인해서 다시 원래 크기로 돌아간다.

4) 닫힘 연산
== 팽창 연산을 먼저 수행하고, 다음으로 침식 연산을 수행한다.
== 팽창 연산으로 객체가 확장되어서 객체 내부의 빈 공간이 메워진다.
== 다음으로 침식 연산으로 확장되었던 객체의 크기가 원래대로 축소된다.
