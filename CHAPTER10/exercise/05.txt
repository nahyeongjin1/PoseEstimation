5. k-NN 알고리즘을 사용하기 위한 OpenCV에서 제공하는 ml 라이브러리에서 학습과
   분류에 사용되는 내부 메서드들과 매개 변수들에 대해서 설명하시오.

ml 모듈의 KNearest_create() 함수로 knn 객체를 생성한다.

학습 : train()
== 첫 번째와 마지막 인수로 학습 데이터와 레이블 행렬을 입력하며, 두 번째 인수로 학습 데이터의 저장 방향을 지정한다.
== 한 행이 하나의 학습 데이터가 되면, cv2.ml.ROW_SAMPLE로 지정하고, 반대의 경우에는 cv2.ml.COL_SAMPLE로 지정한다.

분류(예측) : findNearest()
== 분류하고자 하는 새 샘플과 이웃 샘플의 개수를 인수로 입력
== return (ret, resp, neig, dist) (참거짓, 예측결과 행렬, 이웃 원소 행렬, 새 샘플과 이웃 샘플 거리값 행렬)
